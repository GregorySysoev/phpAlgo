# Шейкерная сортировка (англ. Cocktail sort) 

Также называемая двунаправленная сортировка. Разновидность [пузырьковой сортировки](../bubble/README.md).

Сложность алгоритма:
- Лучший случай: O(n)
- Среднее: O(n^2)
- Худший случай: O(n^2)

Затраты памяти: O(1)
